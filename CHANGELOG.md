# Changelog

All notable changes to **Practice Desk** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres
to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-02-18

### Added

- Project infrastructure: README, CHANGELOG, ROADMAP, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY
- CI/CD: GitHub Actions with auto-issue creation on failure and auto-close on success
- Linting: markdownlint, shellcheck, prettier
- Release automation: commit-and-tag-version with Handlebars changelog templates
- Conventional Commits enforcement via gitlint
- Makefile with colored output and professional targets
